# Inleiding tot het programmeren van Android-tablets met Android Studio (2016)

👉 **Bijbehorende cursus:**
[Inleiding tot het programmeren van Android-tablets met Android Studio (2016)](https://stahe.github.io/nl-android-aout-2016/)

---

## 📖 Overzicht

Dit document is een herschrijving en samenvoeging van verschillende leermiddelen over Android-ontwikkeling. Het is bedoeld om een **stapsgewijze en gestructureerde inleiding** te bieden tot het programmeren van Android-apps, met name voor tablets.

Het bevat een aantal belangrijke ontwikkelingen:

* Invoering van **Android Studio** als belangrijkste ontwikkelomgeving
* Gebruik van het **Gradle**-buildsysteem
* De introductie van **RxJava** voor het beheer van asynchronisme
* Geavanceerde structurering van Android-apps (fragmenten, levenscyclus, client/server-architectuur)

Dit document onderscheidt zich met name door de presentatie van een **generiek raamwerk voor een client/server-Android-app**, dat in talrijke projecten kan worden hergebruikt.



---

## 🎯 Leerdoelen

* De basisprincipes van Android-ontwikkeling begrijpen
* Navigatie tussen weergaven en fragmenten onder de knie krijgen
* De levenscyclus van Android-componenten beheren
* Client/server-architecturen implementeren
* Gegevens bewerken via gebruikersinterfaces
* Asynchrone programmering met RxJava introduceren
